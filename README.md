Jogo de luta em turno (ABS)

OBJETIVO:
assar a sensação do ritmo de uma luta
com movimentação, ataque e defesa, antecipação e gingado.

CONTROLES:
Setas: movimentar, escolher entre movimentos
Z, seta: mudar base
A, seta: mudar guarda

OBS:
alfa 4.0
